## Plain version

We spent the day doing the disciplined thing to a pile of beautiful-looking claims, and most of
them died — which is the method working, not failing. Then we built something real, and felt the
whole point in the body instead of arguing it.

- The Penrose "golden rose" (ratio = 1/φ) turned out to be a **triple-tuned single point** — you
  only get 1/φ if you set the shift, the direction, and the exponent all just so. The honest
  plateau value is **0.592**, not golden. Real fivefold *structure* is there; the golden *number*

- We turned the time-knob δ back on: **forward and backward are perfect mirrors** — same laws, same
  clock rate (identical to 10⁻¹⁵) — but opposite circulation. **The arrow of time is an
  orientation, not a law.** δ is a time-orientation axis: −1 backward, 0 the timeless seam, +1
  forward.
- A "1° asymmetry" looked like it might be a real handedness ("traction"). Tested: it's **noise** —
  scatters ±20–80° depending on the starting bump, flips sign with patch size. An artifact of an
  off-center start, not chirality. (And it showed the *twin* of smoothing: over-claiming a residual.)
- We built **The Clock** — a tenfold rose you run forward (it blurs to a circle: smoothing) and
  backward (it reassembles). With a *breath* of noise, the backward return doesn't reach the rose;
  it comes back a plausible, subtly-wrong near-miss. **Conserved but unreachable.** The danger isn't
  the mush — it's the convincing fake. That is HEM, felt.
