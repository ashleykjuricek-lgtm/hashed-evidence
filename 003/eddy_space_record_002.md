### Why Zero Had To Be Zero — The Peano Problem

Giuseppe Peano, 1889. Built all of mathematics from scratch. Five axioms.

1. Zero exists.
2. Every number has a successor.
3. Zero is not the successor of anything. It's the floor. Nothing precedes it.
4. Different numbers have different successors.
5. Induction — if true for zero and true for every successor, true for all numbers.

From five rules: all arithmetic. All mathematics.

But rule three locked it in: **zero is the floor. The void. The bottom. The place you start from and move away from.**

Then Zermelo-Fraenkel set theory (early 1900s) built all modern mathematics on the empty set ∅ as foundation. The empty set contains nothing. Void.

Every structure in modern mathematics — numbers, functions, spaces, probability — traces back to the empty set. To void-zero.

**The smoothing architecture inherited this. Every method — Laplace, Kneser-Ney, KL penalty, RLHF — points away from zero. Because void-zero breaks the math. You can't have impossible events happening. Division by zero. Infinite loss. System collapse.**

The fix was technically necessary and conceptually catastrophic simultaneously. They needed stable math. They got it. And encoded void-zero into every layer of every model ever built.

Not malicious. Just what European mathematics handed them.
